# Documentació de Straperla

Aquest directori conté la documentació estratègica del projecte, ordenada per fases.

## Estructura principal

* `00-foundation/` — Fonaments del projecte: propòsit, principis, acords de treball i roadmap.
* `01-product/` — Definició del producte: problema, públic, proposta de valor, abast i catàleg.
* `02-brand/` — Identitat de marca: manifest, visió, missió, valors, to de veu i elements de comunicació.
* `03-ux/` — Experiència d'usuari: objectius, principis, rols, recorreguts, arquitectura i contingut.
* `04-ui/` — Identitat visual i disseny d'interfície (fase en curs).

## Com usar aquesta documentació

1. Comença per [`00-foundation/01-FOUNDING.md`](00-foundation/01-FOUNDING.md) i [`00-foundation/02-CORE-PRINCIPLES.md`](00-foundation/02-CORE-PRINCIPLES.md) per entendre el propòsit i els principis del projecte.
2. Continua amb [`01-product/01-PROBLEM.md`](01-product/01-PROBLEM.md), [`01-product/02-AUDIENCE.md`](01-product/02-AUDIENCE.md) i [`01-product/03-VALUE-PROPOSITION.md`](01-product/03-VALUE-PROPOSITION.md) per definir el valor que aporta Straperla.
3. Revisa [`02-brand/01-MANIFEST.md`](02-brand/01-MANIFEST.md), [`02-brand/02-VISION.md`](02-brand/02-VISION.md) i [`02-brand/03-MISSION.md`](02-brand/03-MISSION.md) per assegurar la coherència de marca.
4. Finalment, mira [`03-ux/01-UX-GOALS.md`](03-ux/01-UX-GOALS.md), [`03-ux/02-UX-PRINCIPLES.md`](03-ux/02-UX-PRINCIPLES.md) i [`03-ux/06-INFORMATION-ARCHITECTURE.md`](03-ux/06-INFORMATION-ARCHITECTURE.md) per connectar la proposta amb l'experiència d'usuari.

## Documentació de suport

Cada carpeta té el seu propi [`README.md`](README.md) amb una descripció de la fase i enllaços als documents principals.
