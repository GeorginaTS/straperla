# 🔄 User Flows

Aquest document defineix els principals fluxos d'interacció de Straperla.

Cada flux descriu el recorregut més senzill per aconseguir un objectiu concret.

L'objectiu és reduir al mínim el nombre de passos necessaris perquè l'experiència sigui clara, ràpida i intuïtiva.

---

# Principis

* Cada flux respon a un únic objectiu.
* Prioritzem el menor nombre de passos possible.
* Cada acció ha d'aportar valor.
* Els fluxos estan pensats principalment per a dispositius mòbils.

---

# Descobrir un producte

## Objectiu

Trobar un producte fresc i de proximitat.

```text
Inici
   │
   ▼
Explorar o cercar
   │
   ▼
Seleccionar un producte
   │
   ▼
Consultar la fitxa del producte
   │
   ▼
Conèixer el productor
   │
   ▼
Contactar
```

# Registrar-se com a productor

## Objectiu

```text
Inici
   │
   ▼
Crear un compte
   │
   ▼
Escollir el rol de productor
   │
   ▼
Completar la informació bàsica
   │
   ▼
Sol·licitar la verificació
```

# Publicar un producte

## Objectiu

Donar visibilitat a un producte en pocs minuts.

```text
Panell del productor
   │
   ▼
Nou producte
   │
   ▼
Completar la informació mínima
   │
   ▼
Publicar
   │
   ▼
Producte visible
```
