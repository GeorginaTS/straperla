# 🛤️ User Journeys

Aquest document recull els principals recorreguts que les persones poden fer dins de Straperla.

Els recorreguts es defineixen a partir dels objectius que volen aconseguir, no del seu rol.

---

# Principis

Cada recorregut ha de ser:

* Clar.
* Curt.
* Intuïtiu.
* Accessible.
* Centrat en una única necessitat.

---

# Descobrir un producte

## Objectiu

Trobar un producte fresc, disponible i de proximitat.

## Recorregut

Inici

→ Explorar o cercar

→ Descobrir un producte

→ Consultar la fitxa

→ Conèixer el productor

→ Contactar

---

# Descobrir un productor

## Objectiu

Conèixer una persona o projecte productor.

## Recorregut

Inici

→ Cercar o explorar

→ Consultar el perfil

→ Descobrir els seus productes

→ Contactar

---

# Publicar el primer producte

## Objectiu

Donar visibilitat a un producte de manera ràpida.

## Recorregut

Registrar-se com a productor

→ Completar la informació mínima

→ Publicar el producte

→ Revisar la fitxa publicada

---

# Actualitzar un producte

## Objectiu

Mantenir la informació actualitzada.

## Recorregut

Accedir al perfil

→ Seleccionar un producte

→ Editar la informació

→ Guardar els canvis

---

# Gestionar la disponibilitat

## Objectiu

Mostrar únicament els productes disponibles.

## Recorregut

Accedir al producte

→ Actualitzar la disponibilitat

→ Desar

---

# Principi de disseny

Cada recorregut ha d'ajudar la persona a arribar al seu objectiu amb el mínim nombre de passos necessaris.

---

> *L'experiència no es mesura pel nombre de pantalles, sinó per la facilitat d'aconseguir un objectiu.*
