# Fundació

Aquesta carpeta recull els documents que defineixen els fonaments del projecte.

## Documents

* [`01-FOUNDING.md`](01-FOUNDING.md) — Per què existeix Straperla i quin és el seu propòsit.
* [`02-CORE-PRINCIPLES.md`](02-CORE-PRINCIPLES.md) — Principis que guien totes les decisions del projecte.
* [`03-ROADMAP.md`](03-ROADMAP.md) — Milestones i planificació del projecte.
* [`04-WORKING-AGREEMENT.md`](04-WORKING-AGREEMENT.md) — Acords de treball i criteris de decisió.
* [`05-PARKING-LOT.md`](05-PARKING-LOT.md) — Idees i funcionalitats posades en espera.

## Com usar aquests documents

1. Comença per `01-FOUNDING.md` per entendre el propòsit.
2. Revisa `02-CORE-PRINCIPLES.md` per conèixer els criteris de decisió.
3. Consulta `03-ROADMAP.md` per veure com s'organitzen les fases.
4. Fes servir `04-WORKING-AGREEMENT.md` per alinear la forma de treballar.
5. Desa idees no prioritàries a `05-PARKING-LOT.md`.
