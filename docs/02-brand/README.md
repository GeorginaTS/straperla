## Objectiu de la fase

Abans de dissenyar una marca, hem de poder respondre amb claredat aquestes preguntes:

* Qui som?
* Per què existim?
* En què creiem?
* Quin compromís assumim?
* Com volem que les persones ens recordin?

---


> *Una marca no és el que diem que som. És el que les persones recorden de nosaltres.*
